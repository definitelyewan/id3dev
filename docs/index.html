ID3dev

ID3dev is a low-level library for reading, manipulating, and writing ID3 metadata of any version.

View Code Get Started

Progress

As of right now, ID3dev can read, write, and manipulate ID3v1, ID3v1.1, ID3v2.2, ID3v2.3, and ID3v2.4 tags. However, These standards have not been unified into single functions able to handle all mentioned standards. Here is a breakdown of the current status of this project.
