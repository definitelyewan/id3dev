◆ id3ConvertId3v1ToId3v2()

bool id3ConvertId3v1ToId3v2 ( ID3 *  metadata)

Converts the stored ID3v1 tag to an ID3v2 tag to be held in the ID3 structre. The prefered standard will be used to infer what version of ID3v2 is to be used but, if the standard is ID3V1 this function will fail. However, if there is a stored ID3v2 tag it will be removed and replaced. On success this function will return true, otherwise it will return false.
